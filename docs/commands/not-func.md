# Not: `!`

> Reads the stdin and exit number from previous process and not's it's condition

## Description

Reads the stdin and exit number from previous process and not's it's condition.

## Usage

```
<stdin> -> ! -> <stdout>
```

## Examples

### Inverting true

```
» echo "Hello, world!" -> !
false
```

### Inverting false

```
» false -> !
true
```

## Synonyms

* `!`
* `not`
