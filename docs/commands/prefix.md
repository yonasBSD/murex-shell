# Add Prefix: `prefix`

> Prefix a string to every item in a list

## Description

Takes a list from stdin and returns that same list with each element prefixed.

## Usage

```
<stdin> -> prefix str -> <stdout>
```

## Examples

```
Â» ja [Monday..Wednesday] -> prefix foobar
[
    "foobarMonday",
    "foobarTuesday",
    "foobarWednesday"
]
```
