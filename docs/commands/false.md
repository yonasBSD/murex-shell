# False: `false`

> Returns a `false` value

## Description

Returns a `false` value.

## Usage

```
false -> <stdout>
```

## Examples

### No flags

By default, `false` also outputs the term "false":

```
» false
false
```

### Silent

You can suppress that with the silent flag:

```
» false -s
```

## Flags

* `-s`
    silent - don't output the term "false"
