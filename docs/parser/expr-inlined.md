# ( expression )

> Inline expressions

## Description

Expressions can be inlined via parenthesis. For example:

```
» echo (1 + 2)
3
```

The opening brace has to be the first character of that parameter, otherwise
that parameter will be treated like a string.

```
» echo 1+(1 + 2)
1+(1 + 2)
```
